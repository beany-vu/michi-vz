<h2 className="mv-h2">Shared theming — MichiVzProvider</h2>
<p className="mv-lead">
  Wrap charts in <code>MichiVzProvider</code> to give a label the same colour
  everywhere, and to share highlighted / disabled state across a dashboard.
</p>

```jsx
import { MichiVzProvider, RibbonChart } from "michi-vz";

<MichiVzProvider colorsMapping={{ Coal: "#6b7280", Gas: "#f59e0b", "Wind & Solar": "#22c55e" }}>
  <RibbonChart series={series} keys={keys} width={900} height={480} margin={margin} />
</MichiVzProvider>
```
